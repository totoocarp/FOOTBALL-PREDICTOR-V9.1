# Disabled modules for the V9.0 Beta

The V9.0 Beta is focused on model training, data collection, validation, learning, statistics, diagnostics, and database review.

Public/end-user modules are intentionally hidden from the main web interface until the project has enough real learned matches to justify reopening them. The code and templates are kept here so they can be restored in a future post-beta release without losing functionality.

Disabled public modules:

- Home / marketing dashboard
- Manual predictor UI
- World Cup UI
- AI panel
- Configuration UI

Internal APIs and services may still exist where they support training, scheduled predictions, validation, or future releases.
